pi · companion for pi-subagents

pi-subagent-toolbudget

Caps a sub-agent's tool calls — soft nudge, then hard stop — so a runaway agent writes its final report instead of overflowing context. No-op outside sub-agents.
sub-agent: reviewer  ·  [[toolbudget: soft=40 hard=60]]
allow  #1   read  OrderService.java
allow  #2   grep  "resolveTotal"
      ⋮
allow  #40  read  ← soft: start writing your final report
allow  #60  ls
BLOCK  #61  read  ← hard stop · report/output tools stay open
→ sub-agent delivers its report  (exit 0, not a silent exit 1)
